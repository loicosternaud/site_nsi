Voici les projets que j'ai réalisés en NSI sur le WEB :

-- Vous êtes parti en randonnée dans une forêt mystérieuse, malheureusement vous avez perdu le lieu du campement où se trouve la cabane pour dormir. Si vous voulez faire l'aventure du livre dont vous êtes le héros, il faut cliquer sur Recherche de la cabane.
-- Le deuxiéme projet était un formulaire qui permettra de générer et d’envoyer des messages à vos camarades de classe. Si vous voulez envoyer un message, il faut cliquer sur Messagerie.
-- Les messages envoyés pourront être lus ensuite supprimés, dans l’espace Boite Aux Lettres de chacun, sur le site https://nsibranly.fr/. Si vous voulez voir votre boîte aux lettres, il faut cliquer sur Boîte aux lettres.
-- Pour le troisième projet, j'ai découvert le langage de programmation JavaScript qui nous a servi à calculer le prix TTC avec le prix HT et la TVA. Si vous voulez tester le calculateur de prix TTC, il faut cliquer sur Calcul prix TTC.
On a continué le JavaScript avec plusieurs petits exercices,
-- le premier était de créer une page web interactive sur ordinateur ou smartphone, qui vous demande votre nom et l'affiche par la suite. Si vous voulez tester la page interactive, il faut cliquer sur monMonde.
-- le deuxième était de créer une page web interactive du type « convertisseur de monnaie », il s'agit d'une page web où vous pouvez convertir une somme en euros en dollars US ou Yuans chinois. Si vous voulez tester le convertisseur, il faut cliquer sur Convertisseur.
-- le troisième était de créér une animation sur une page Web. Après click sur la flèche gauche ou droite, le personnage se déplace dans la fenêtre. Si vous voulez tester les animations, il faut cliquer sur Desert.
